A modern, feature-rich music app built with Expo and TypeScript, designed for seamless music playback, playlist management, and YouTube integration.

![Home Screen](assets/home_screen.webp)

## Features

- **Local Music Playback**: Play your local music files with a sleek audio player interface.
- **Playlist Management**: Create, edit, and manage your playlists effortlessly.
- **YouTube Integration**: Download and play music from YouTube directly in the app.
- **Recently Played**: Keep track of your listening history.
- **Cross-Platform**: Runs on iOS, Android, and Web using Expo.
- **Offline Support**: Access your local music without an internet connection.
- **Beautiful UI**: Modern design with intuitive navigation and controls.

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/lincolnpaul-devEng/Muzik.git
   cd Muzik
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Run on your device**:
   - For Android: `npm run android`
   - For iOS: `npm run ios`
   - For Web: `npm run web`

## Usage

- **Home Screen**: Browse your music library and recently played tracks.
- **Library**: Manage your playlists and local music.
- **Download**: Search and download music from YouTube.
- **Settings**: Customize app preferences.
